## 4. ROI指標 — 投資対効果の測定

### A. 時間投資と削減効果

| 項目 | 推定値 | 根拠 |
|---|---|---|
| **日次作業時間** | 2〜3時間 | X投稿1本（30分） + Note執筆（1.5〜2時間） + 運営（30分） |
| **AI活用による削減** | 60〜90分/日 | Claude AI：リサーチ・草稿（60分）、ChatGPT：アイデア出し（30分） |
| **削減率** | 30〜40% | 本業に使える時間を1日1時間確保 |
| **月間削減** | 20〜30時間 | 約1.5営業日分 |
| **年間削減** | 240〜360時間 | 約1ヶ月分の工数 |

### B. ROI計算式

```
ROI = （収益 - コスト） ÷ コスト × 100%
```

**目標ROI基準:**
- Month 1-2: マイナス（投資フェーズ）
- Month 3-5: 0% 以上（損益分岐点達成）
- Month 6+: 100% 以上（自走段階）

**定性評価基準:**
- ⭐ X フォロワー増、Note読者数増、有料受注発生
- ⭐ 営業本業のナレッジ化・ドキュメント化の時間削減
- ⭐ AI社員の活用度向上（プロンプト精度・ワークフロー最適化）

## 5. コスト管理ルール

### 5.1 月次記録（毎月1日）

```
□ 前月の実績（収支・稼働日数・フォロワー数）を記録する
□ 今月のコストを確定する
□ ROI を計算し、Baseline達成度を確認する
□ ツール追加・削除の検討が必要か判定する
```

**記録場所:** `/reports/YYYY-MM-DD-monthly-financial-review.md`

### 5.2 ツール追加時のチェック

新規ツール導入判定フロー：

```
1. 月額コストを確認
   ↓
2. 月収の3倍以上が見込めるか？
   → YES: 追加検討OK
   → NO: 見送り
   ↓
3. 既存ツールで代替できないか？
   → YES: 検討中止
   → NO: 追加決定
   ↓
4. コスト管理.md に記録
```

### 5.3 ツール削除基準

以下のいずれかに該当する場合、削除を検討：

- **未使用状態が2ヶ月以上続いている**
- **月額コスト > その月の削減効果（時間 × 時給換算）**
- **後続ツールが同機能をカバーしている**

### 5.4 閾値管理

| 指標 | 閾値 | アクション |
|---|---|---|
| 月収 / 月コスト | 3未満 | ツール見直し・コスト削減検討 |
| 月収 / 月コスト | 3以上 | 正常範囲・ツール追加検討可能 |
| 赤字期間 | 3ヶ月超 | ビジネスモデル見直し・プラン変更 |
| ROI | -100% 以上 | 継続投資許容 |
| ROI | -100% 未満 | 事業計画再検討 |
