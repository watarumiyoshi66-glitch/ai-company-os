# Weekly Note Candidates SOP

## 目的

平日に蓄積した調査・X投稿案・Services×GenAI素材から、週1回Note候補を3本に整理する。

## 実行タイミング

- 推奨：毎週金曜または日曜
- 実行主体：Claude Code Remote
- 仕上げ：CodexがNote化優先度を評価
- 保存先：`/content/note/research/YYYY-MM-DD-weekly-note-candidates.md`

## 入力

- `/research/market/`
- `/research/competitors/`
- `/content/x/drafts/`
- `/content/x/performance/`
- `/intel/services-genai/`
- `/reports/weekly-codex-audit-template.md`

## 出力

- Note候補3本
- それぞれの読者、悩み、約束、根拠
- 無料/有料の推奨
- CTA候補
- オーナー判断事項

## Codex評価観点

| 観点 | 確認内容 |
|---|---|
| 読者価値 | 読者の悩みが明確か |
| 独自性 | 外資IT営業の視点が入っているか |
| 根拠 | 出典や経験があるか |
| 収益性 | 有料化・テンプレ・ソフト導線があるか |
| 書きやすさ | 1週間以内に書けるか |

## 完了条件

- 候補3本を保存済み
- 推奨1本を明記済み
- オーナー判断事項を `tasks/owner-decisions.md` に追記済み
