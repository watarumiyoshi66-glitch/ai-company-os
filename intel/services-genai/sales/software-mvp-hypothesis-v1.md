# Software MVP Hypothesis v1 — Sales GenAI Ops Dashboard

目的：有料Noteで配布したテンプレを「毎日回す」ための最小ソフトウェア。

---

## 誰のため

- 営業マネージャー / RevOps（現場運用を回す人）

## 最小機能（MVP）

1. 役職（AI社員）定義：担当と責務を1画面で管理
2. SOPチェックリスト：今日の実行（やる/やらない）を記録
3. 計測ログ：毎日の入力（数値/未確認/メモ）を残す
4. レビュー：1週間の“詰まり”を抽出（どこが止まったか）

## 価値仮説

- “PoC疲れ”の原因である「運用不在」「評価不在」を最小コストで解消できる

## 最短検証

- 有料Note購入者にテンプレ配布→2週間運用→詰まりTop10を収集→MVP要件に確定
