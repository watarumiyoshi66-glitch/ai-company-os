# AI Company OS

> An AI-native operating system for a one-person enterprise business.

---

## Overview

This repository runs the full stack of an AI-powered solo company:
content creation, research, sales knowledge, and agent orchestration — all in one place.

---

## Structure

```
ai-company-os/
├── /company        # Business strategy, positioning, goals
├── /agents         # AI agent definitions and prompts
├── /content        # X posts, Note drafts, content calendar
├── /memory         # Reusable knowledge base
├── /workflows      # Recurring task flows and SOPs
├── /research       # AI, enterprise IT, market insights
└── /reports        # Weekly reviews and performance logs
```

---

## Core Functions

| Function | Description |
|---|---|
| Content Engine | X posts and Note articles on enterprise AI |
| Knowledge Base | Accumulated sales and strategy insights |
| Agent Roster | AI agents acting as team members |
| Workflow Library | Reusable automation templates |
| Research Hub | Curated intelligence on AI trends |

---

## Principles

- **AI-first** — every workflow runs through AI before human review
- **Japanese-first** — primary output language is Japanese
- **Never auto-publish** — all external posts require human approval
- **Compounding** — every output feeds back into `/memory`
