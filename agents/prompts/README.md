# AI社員プロンプト運用ガイド

このフォルダは、AI社員ごとの実行プロンプトを保管する場所です。

## 使い方

1. 依頼したい社員のプロンプトを開く
2. 「依頼テンプレート」に今回の目的・素材・制約を入れる
3. 出力後、社員自身に「自己採点」と「改善版」を出させる
4. 必要に応じて他の社員へレビューを回す

## 共通ルール

- 事実と解釈を分ける
- 出典・日付・前提条件を明記する
- オーナーが判断すべきことを先に出す
- 機密情報・個人情報・顧客が特定される情報は書かない
- 最後に必ず自己採点を付ける

## 自己採点フォーマット

```markdown
## 自己採点

| 評価軸 | 点数 | 理由 |
|---|---:|---|
| 目的適合 | /10 |  |

- 
```

## 品質ゲート

合計40点未満の成果物は提出前に自分で修正します。
事実確認が弱い場合は「未確認」と明記し、断言しません。
