# Owner Decisions

Claude Code Remote と Codex が自動運営の中で見つけた、オーナー判断が必要な事項を集約する。

## 運用ルール

- AIが勝手に決めてはいけない事項だけを書く
- 推奨案と理由を必ず添える
- 判断後は「決定内容」と「反映先」を追記する
- 公開、投稿、送信、価格変更、個人情報利用、外部連携は必ずここに記録する
